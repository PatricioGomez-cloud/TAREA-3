## Descripción

Esta página presenta una pieza narrativa basada en el análisis de la evolución del valor de mercado de los clubes de la Primera División chilena entre 2019 y 2026. A través de una visualización de datos y una crónica breve, se explora la diferencia económica entre los equipos y su posible relación con el rendimiento deportivo.

## Proceso de trabajo

Para esta tarea utilicé una visualización desarrollada durante el proyecto grupal sobre el fútbol chileno. A partir de ella construí una historia enfocada en la evolución del valor de mercado de los clubes y la brecha económica existente entre los equipos con mayores recursos y el resto del campeonato.

La página fue desarrollada utilizando HTML, incorporando elementos básicos como encabezados, párrafos, imágenes, enlaces y contenedores para organizar la información. Finalmente, el sitio fue publicado mediante GitHub Pages para que pudiera ser visualizado en línea.
